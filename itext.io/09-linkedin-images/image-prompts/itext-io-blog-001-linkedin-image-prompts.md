# LinkedIn Image Prompt Pack — iText.io (blog-001)

Tool: ChatGPT (GPT image generation). Recommended aspect ratio: landscape, target 1200x627 / 1.91:1 (use the widest landscape option, ~1536x1024, then crop to 1200x627). Generate one image per prompt below.

## v01 — Clean editorial hero image
- **Image objective:** Hero visual on a familiar phone line gaining a text channel.
- **Product/service context:** Text-enabling an existing business number.
- **Audience context:** SMB and customer-facing teams.
- **Visual style:** Clean, modern, optimistic editorial illustration.
- **Composition:** Wide landscape; a single stylized phone/line motif splitting into two clean lanes — a voice lane and a message lane running alongside it; large clean space for a later headline.
- **Color/mood guidance:** Light base, confident blue/teal primary, green "read" accent; modern and positive.
- **Subject matter:** One line branching into parallel voice and text lanes.
- **Restrictions:** No text, no logos, no real UI, no phone numbers, no faces.
- **Negative prompt:** text, words, letters, numbers, logos, watermark, app UI screenshot, faces, clutter, low resolution.
- **Aspect ratio:** 1.91:1 (1200x627 target).
