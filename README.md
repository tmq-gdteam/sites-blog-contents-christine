# Blog Creation Pipeline

Source material and generated blog content for client sites, organized one folder per site.

Each site folder may contain:

- **Website dossier** — structured summary of the site, audience, claims-safety boundaries.
- **Deep research** — supporting research documents (PDF/markdown).
- **Podcast transcript** — the primary source for blog generation.
- **Generated blog posts** — finished editorial articles in Markdown.

## Sites

- `polished.cx/` — real-time voice AI / accent localization
- `noforgery.ai/` — identity verification
- `halfoff.ai/`
- `icalendly/`
- `customers.new/`

`generate-blog-workspace/` holds test/sample material used while building the blog-generation workflow.
